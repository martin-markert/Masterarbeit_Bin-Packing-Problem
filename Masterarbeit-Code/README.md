# Description
This code is the re-implementation of the approach to the 3D bin packing problem as presented by Que et al. in the paper "Solving 3D packing problem using Transformer network and reinforcement learning". Unless otherwise stated, references in the code to chapters and figures refer to the paper. The paper can be found [here](https://doi.org/10.1016/j.eswa.2022.119153)

## Installation
`pip install -r requirements.txt`

## Parameters
Set training parameters at `parameters.py`

## Run
`python train.py`

## Test and see results
`python test.py`
